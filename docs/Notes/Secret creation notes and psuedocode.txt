Simple Cert Creation

Need:
Ansible Template Survey
	cred_name
	cred_user (Default: cred_user }}
	cred_password
	namespace

Inventory:
	Launch from api0
		use the Rancher Inventories
		

Limit:
	No limit needed as part of template if using Rancher Inventory
	
Automation:
temporary file with base64 encoded password created from survey
	lineinfile the password piped to a base64 filter
OR
template that populates the base64 encoded password into a yaml file

Run kubectl command to create credential, using the temp file to provide the base64 password or applying the yaml file.


add_cred.yaml template:

apiVersion: v1
kind: Secret
metadata:
  name: {{ cred_name }}
  namespace: {{ cred_namespace }}
type: Opaque
data:
  cred_user: {{ cred_user }}
  {{ cred_name }}: {{ cred_password | b64encode }}



 
kubectl apply -f add_cred.yaml
